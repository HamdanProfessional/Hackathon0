# In Progress - Cloud - Personal Domain

Personal domain items currently being processed by Cloud VM.

**Purpose:** Temporary storage for personal items that are actively being worked on by Cloud.

**Lifecycle:**
1. Item created in `Needs_Action/Personal/`
2. Cloud claims item → moves here
3. Cloud processes item → moves to `Pending_Approval/` or `Done/Personal/`

**Claim-by-Move:** Moving an item here claims it for Cloud processing.
