# Social Media Manager

Schedule posts, track engagement, analyze performance, and optimize social media presence across multiple platforms.

## Purpose

The Social Media Manager skill **schedules** content for optimal posting times, **tracks** engagement metrics across platforms, **analyzes** content performance, **optimizes** posting strategy based on data, and **maintains** consistent brand presence. This skill **orchestrates** the entire social media workflow from content generation to posting and analytics.

## Design Philosophy

- **Data-Driven**: Use engagement analytics to inform strategy
- **Platform-Specific**: Optimize for each platform's unique characteristics
- **Consistent Brand**: Maintain voice and visual identity across platforms
- **Engagement-Focused**: Prioritize meaningful interactions over vanity metrics
- **Proactive**: Schedule content in advance, react to trends

## Workflow

1. **Review** approved content from `/Approved/`
2. **Schedule** posts based on platform best practices
3. **Post** content via MCP servers at optimal times
4. **Track** engagement metrics (likes, comments, shares)
5. **Analyze** performance and identify trends
6. **Optimize** future content based on data
7. **Report** on performance in weekly briefings

## Modularity

Extensible with:
- Multi-platform scheduling
- A/B testing frameworks
- Influencer collaboration tracking
- Social listening and trend detection
- Automated engagement responses
- Integration with analytics APIs
