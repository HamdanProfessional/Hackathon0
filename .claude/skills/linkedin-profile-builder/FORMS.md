## Application Instructions

### Step 1: Review Drafts
- Read through each suggested update
- Edit to match your voice and experience
- Verify all facts and figures are accurate

### Step 2: Apply to LinkedIn

1. **Headline:**
   - Go to your LinkedIn profile
   - Click the pencil icon next to your headline
   - Paste the suggested headline
   - Save changes

2. **About Section:**
   - Scroll to About section
   - Click the pencil icon
   - Delete existing content
   - Paste the suggested about section
   - Review formatting
   - Save changes

3. **Experience:**
   - For each position, click the pencil icon
   - Update the description with suggested content
   - Add any missing achievements
   - Save changes

4. **Skills:**
   - Go to Skills section
   - Click "Add skill"
   - Add each recommended skill
   - Remove outdated skills if needed

### Step 3: Verify
- View your profile as public
- Check for formatting issues
- Verify all links work
- Ask a colleague for feedback
