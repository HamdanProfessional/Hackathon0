## Template Variables Reference

| Variable | Type | Description |
|----------|------|-------------|
| `profile_id` | string | LinkedIn profile ID (e.g., "hamdan-mohammad-922486374") |
| `name` | string | Full name from profile |
| `headline` | string | Professional headline |
| `location` | string | Geographic location |
| `connections_count` | int | Number of connections |
| `completeness_score` | int | 0-100 score |
| `headline_score` | int | 0-10 score for headline |
| `about_score` | int | 0-20 score for about section |
| `photo_score` | int | 0-10 score for profile photo |
| `experience_score` | int | 0-25 score for experience |
| `education_score` | int | 0-10 score for education |
| `skills_score` | int | 0-15 score for skills |
| `recommendations_score` | int | 0-10 score for recommendations |
| `about_word_count` | int | Number of words in about section |
| `experience_count` | int | Number of experience entries |
| `education_count` | int | Number of education entries |
| `skills_count` | int | Number of skills |
| `recommendations_count` | int | Number of recommendations |
| `posts_count` | int | Number of recent posts |
| `has_photo` | bool | Whether profile has a photo |
| `strengths` | list | List of strength strings |
| `weaknesses` | list | List of weakness strings |
| `suggestions` | list | List of suggestion objects |
| `primary_keywords` | list | List of primary SEO keywords |
| `secondary_keywords` | list | List of secondary SEO keywords |
| `timestamp` | string | ISO 8601 timestamp |
| `extraction_date` | string | Human-readable date |
| `screenshot_path` | string | Path to screenshot file |
