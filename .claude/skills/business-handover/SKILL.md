# Business Handover (Monday Morning CEO Briefing)

## Overview

Generate comprehensive Monday Morning CEO Briefing that autonomously audits business performance, reviews logs, analyzes bank transactions, compares progress to targets, and reports revenue and bottlenecks. This is the **standout feature** of the AI Employee system.

## Purpose

The Business Handover skill **transforms** the AI from a reactive chatbot into a **proactive business partner**. Every Monday morning, it autonomously audits the business, analyzes performance, and presents actionable insights to the CEO.

## Key Features

- **Autonomous Execution**: 7-task workflow runs automatically (Ralph Wiggum loop)
- **Business Audit**: Scans logs, transactions, and completed work
- **Performance Analysis**: Compares actual vs. targets
- **Bottleneck Detection**: Identifies delays and inefficiencies
- **Proactive Suggestions**: Surfaces optimization opportunities
- **Revenue Tracking**: Monitors income and expenses
- **Action List**: Prioritized tasks for the week

## Workflow

### Ralph Wiggum Autonomous Execution

```
1. Check Gmail for urgent weekend messages
2. Review calendar for updated events
3. Analyze business performance from logs
4. Compare progress to business targets
5. Generate proactive optimization suggestions
6. Create professional CEO briefing document
7. Create prioritized action list for the week
```

### Trigger

**Scheduled**: Every Monday at 7:00 AM (via PM2 cron job)

**Manual**: Run on-demand using `/ralph-loop` command

### Output Files

```
AI_Employee_Vault/Briefings/
├── YYYY-MM-DD_Monday_Briefing.md         # CEO briefing document
└── YYYY-MM-DD_Monday_Actions.md          # Prioritized action list
```

## Integration with Ralph Wiggum

The Business Handover uses Ralph Wiggum for autonomous execution:

```
1. Start Ralph loop
2. Execute 7 tasks sequentially
3. Continue until all tasks complete
4. Generate briefing document
5. Create action list
6. Exit when complete
```

### Ralph Command

```bash
# Start Ralph loop for CEO Briefing
/ralph-loop "Generate Monday Morning CEO Briefing with business audit, performance analysis, and proactive suggestions" \
  --completion-promise "BRIEFING_COMPLETE" \
  --max-iterations 20
```

## Troubleshooting

### Briefing Not Generated

**Problem**: No briefing file created

**Solution**:
1. Check PM2 logs: `pm2 logs daily-briefing`
2. Verify Ralph skill is installed: `/ralph-loop --help`
3. Check Business_Goals.md exists
4. Verify vault path is correct

### Incomplete Analysis

**Problem**: Briefing missing sections

**Solution**:
1. Check Logs/ folder has data
2. Verify Accounting/ folder exists
3. Check Done/ folder has completed tasks
4. Review Ralph loop output

### Performance Issues

**Problem**: Briefing takes too long

**Solution**:
1. Reduce data scope (current week only)
2. Optimize log parsing
3. Cache frequently accessed data
