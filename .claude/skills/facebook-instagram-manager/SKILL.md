## üìã WORKFLOW

### 1. Content Generation

**Input:** Topic or key message
**Process:**
1. Understand the purpose and target audience
2. Generate platform-specific content:
   - **Facebook:** Longer form, more descriptive, link-friendly
   - **Instagram:** Visual-first, hashtags, shorter captions
3. Create approval file in `/Pending_Approval/`
4. Wait for human approval

**Output:** `META_POST_YYYYMMDD_HHMMSS.md` in `/Pending_Approval/`

### 2. Approval Process

**Process:**
1. User reviews content in `/Pending_Approval/`
2. User makes edits if needed
3. User moves file to `/Approved/`
4. `meta_approval_monitor.py` detects the file
5. Posts to Meta Business Suite (Facebook + Instagram)
6. Moves to `/Done/`
7. Generates performance summary in `/Briefings/`

### 3. Performance Tracking

**Process:**
1. Monitor engagement metrics (likes, comments, shares)
2. Generate weekly summaries
3. Suggest content improvements
4. Track best posting times

## üé® CONTENT GENERATION RULES

### Facebook Content Style

- **Length:** 100-300 characters optimal
- **Tone:** Conversational, informative
- **Links:** Always include for website/posts
- **Emojis:** Use sparingly (1-2 per post)
- **Hashtags:** 3-5 relevant hashtags
- **Call-to-action:** Clear CTA in every post

**Example:**
```
Excited to announce our latest product update! üöÄ

We've been working hard to bring you new features based on your feedback. Check out what's new:

[Link to blog post]

What feature are you most excited about? Let us know in the comments! üëá

#ProductUpdate #NewFeatures #CustomerFeedback
```

### Instagram Content Style

- **Length:** 50-150 characters (shorter than FB)
- **Tone:** Visual, engaging, emoji-friendly
- **Visuals:** ALWAYS include image/video reference
- **Hashtags:** 10-30 hashtags (mix of popular + niche)
- **Stories:** Mention if suitable for Stories
- **Call-to-action:** Swipe up / Link in bio

**Example:**
```
New product alert! üöÄ‚ú®

We've been listening to your feedback and our latest update is HERE.

Check out the link in our bio to see what's new! üì±

.
.
#tech #startup #productlaunch #newfeatures #innovation #startuplife #technews #productupdate #customerlove #feedback #launch #news #excited #teamproud
```

## üéØ BEST PRACTICES

### DO ‚úÖ
- Create platform-specific content (don't just copy-paste)
- Use high-quality images for Instagram
- Include clear calls-to-action
- Post consistently (3-5 times per week)
- Respond to comments within 24 hours
- Use analytics to refine strategy
- Test different content types

### DON'T ‚ùå
- Don't post the exact same content to both platforms
- Don't use all hashtags on Facebook (save for Instagram)
- Don't ignore engagement (comments, messages)
- Don't post at random times (use optimal scheduling)
- Don't buy followers or engagement
- Don't over-post (max 2-3 per day per platform)

### Human-in-the-Loop
- ALL posts require human approval before publishing
- Approval file must be manually moved to `/Approved/`
- No automated posting without review

### Content Review
- Check for brand consistency
- Verify facts and claims
- Ensure compliance with platform terms
- Review for sensitivity/appropriateness

- If posting fails, file stays in `/Approved/` for retry
- Logs all actions to daily log file
- Generates error summaries for troubleshooting

## üìö REFERENCE MATERIAL

### Meta Business Suite
- **URL:** https://business.facebook.com/latest/composer
- **Features:** Schedule posts, manage both platforms, analytics
- **Required:** Facebook Page + Instagram Business Account

### Best Posting Times (General)
- **Facebook:** Weekdays 9am-3pm
- **Instagram:** Weekdays 11am-1pm, evenings 7-9pm
- **Best Days:** Tuesday, Wednesday, Thursday

### Platform-Specific Tips
- **Facebook:** Prioritize shares and comments over likes
- **Instagram:** Prioritize saves and shares (algorithm signal)
- **Instagram Stories:** Post 3-5 times per day for visibility
